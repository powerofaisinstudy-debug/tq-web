SDCEngine.protect()

Memory-safe path for massive tensors. Uses chunked streaming to keep RAM usage flat even with 1B+ parameters.

from torchquery import SDCEngine

# Handles tensors > 100M elements automatically
clean_tensor = SDCEngine.protect(large_tensor, sigma=10.0)
